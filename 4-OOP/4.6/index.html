    6.Built-in Decorator

    1. @property

    @property 的功能即实现 OOP 中的 setter 和 getter，其实现比较复杂。

    class Rectangle(object):

    def __init__(self, length, width):
        self._length = length
        self._width = width

    @property
    def length(self):
        return self._length

    @length.setter
    def length(self, l):
        if l < 0:
            self._length = 0
            return
        self._length = l

r = Rectangle(1, 2)
r.length = -1
print(r.length)
    • 通常，用装饰函数的名字表示封装的属性名，而用单下划线开头的名字表示成员的名字；
    • 如果只设置 getter 的装饰器，则相应的属性是只读的。

    2. @classmethod 和 @staticmethod

    即静态方法。其中 @classmethod 和 @staticmethod 不同之处在于， @classmethod 的第一个参数默认传入调用这个静态方法的类的引用，可以知道是通过基类还是子类调用的。
