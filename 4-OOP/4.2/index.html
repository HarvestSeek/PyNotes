    2.Class and Instance

    1. 创建类

    在 Python 中，通过 class 关键字定义一个类，类名通常使用大驼峰命名法：

    2. 属性与访问限制

     一般在构造函数 def __init__(self, ...) 中把类的属性绑定到 self 上，也可以在构造函数前声明。上面的例子中，类的成员都是 public 的，可以通过添加下划线实现 private ：

def __init__(self, mushrooms=3):
    self.__name = 'Teemo'
    self.__mushrooms = mushrooms
     尽管变量被设置为私有，但仍可以通过 instance._Teemo__name 的方式访问，Python 并不阻止访问者做坏事。
     在 Python 中，形如 __<arg_name>__ 的变量是特殊变量，可以直接访问；形如 _<arg_name> 的变量代表 “虽然我可以被访问，但请把我视为私有变量，不要随意访问”。

    3. 方法

     类的方法的第一个参数必须显式声明为 self (也可以是 this，但通常用 self)，表示这个对象。
     私有方法也通过添加双下划线前缀实现：

def flash(self):
    print(self._Teemo__name + ' flashed')

def tp(self):
    print(self._Teemo__name + ' transfered')

def run(self):
    print(self._Teemo__name + ' ran')

hero = Teemo(20)

# 动态添加实例属性
hero.enemy = 'Fizz'

# 动态添加类属性
Teemo.partner = 'Tristana'

# 动态添加实例方法
# 方式一: 需要添加参数
hero.flash = flash
hero.flash(hero)

# 方式二
hero.run = types.MethodType(run, hero)
hero.run()

# 动态添加类方法
Teemo.tp = tp
hero.tp()
    此外，还需要注意：
    • 动态添加属性 __<attribute> 是无法作为原类的私有变量；
    • 当类属性和实例属性同名，访问时类属性会覆盖实例属性。

    5. 函数重载

     函数的重载是为了摆脱静态语言对于参数的限制的一种办法 (如 Java)。Python 作为一门动态类型语言，函数的参数没有类型限制且带有默认参数等功能，因而不需要函数的重载。
