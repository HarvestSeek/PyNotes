    4.Special Attributes and Methods

    1. 类的特殊属性

    (1). 基本的专有属性

    属性名 作用
    class/obj.__doc__ docstring
    class.__name__ 类名
    class/obj.__module__ 类/对象所在模块
    obj.__class__ 对象所属的类名
    class.__bases__ 继承的父类
    class/obj.__dict__ 成员字典，存储所有类/实例成员信息

    (2). class.__slots__

    __slots__ 是一个 tuple 形式的类变量，一旦设定以后，obj.__dict__ 就会无效，且此类的实例不能拥有这个 tuple 中以外的属性。它的好处在于：

    • 可以限制不能随意动态绑定属性；
    • tuple 占用内存少于散列表，当实例对象较多，可显著减少内存占用。
    需要注意几点：
    • 每个子类都要定义 __slots__，解释器默认忽略继承的 __slots__；
    • 如果不把 __weakref__ 加入 __slots__，实例就不能作为弱引用的目标。

    (1). 基本的专有方法

    方法名 作用

    __len()__ 使得对象可以直接作用于len()函数
    __str()__ toString()
    __repr()__ 交互环境下的toString()

    (2). __iter()__、__next()__

    如果一个类想被作用于 for-in 循环，就必须实现 __iter()__，该方法会返回一个可迭代对象；随后，for 循环就会不断调用这个对象的 __next()__ 方法取得下一个值。

    class Fibonacci():

    def __init__(self, maxn):
        self.maxn = maxn
        self.pre = 1
        self.curr = 0
        self.cnt = 0

    def __iter__(self):
        return self

    def __next__(self):
        self.cnt += 1
        if self.cnt > self.maxn:
            raise StopIteration
        self.pre, self.curr = self.curr, self.pre + self.curr
        return self.curr

    class Container():

    def __setitem__(self, key, value):
        self.vals[key] = value

    def __getitem__(self, index):
        return self.vals[index]

    class Container():
    ...
    def __getitem__(self, index):
        if isinstance(index, int):
            return self.vals[index]
        if isinstance(index, slice):
            start = index.start
            stop = index.stop
            if start is None:
                start = 0
            if stop is None:
                stop = len(self.vals)
            ret = []
            for x in range(start, stop):
                ret.append(self.vals[x])
            return ret

f = Container()
f[0] = 1
f[1] = 2
print(f[0:2])
    更完整的切片还要处理负数和 step 参数。

    (5). __getattr()__

    当调用某个类的某个属性时，如果这个属性不存在，那么 Python 解释器就会尝试从 __getattr(self, attr)__ 中获得属性。

    class URLGenerator(object):
    """__getattr__ and __call__ return an obj for chaining"""

    def __init__(self, header):
        self.header = header

    def __getattr__(self, path):
        # path is name of attribute (string)
        return URLGenerator('%s/%s' % (self.header, path))

    def __str__(self):
        return self.header

    def __call__(self, string):
        return URLGenerator('%s/%s' % (self.header, string))

    __repr__ = __str__

g = URLGenerator('http://www.abc.xyz.com')
s1 = '4-OOP'
print(g('4-OOP').test)
    可以通过内置函数 callable() 判断一个对象是否为可调用对象。
