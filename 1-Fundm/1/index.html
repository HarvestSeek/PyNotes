    1. basic types

    1. 数值类型

    """ basic types """
# int
var, varb, varo, varh = 1, 0b110, 0o34, 0xaf
# long: 3.0 之后无需显式声明，会自动识别
varl = 0xafb2L
# float
varf = 2. + 3e12
# complex (built-in attribute: real, imag, conjugate)
varc = 1 + 3e10j

""" basic operators """
a = 3 // 2  # quotient
a = 3 % 2   # remainder
a = a ** a  # square

    2. String

    (1). 编码

    • ASCII: 早期编码，包含英文字母、数字、符号等 127 个字符，每个占 1 字节
    • Unicode: 统一编码，但只是规定了编码的二进制数，而未规定如何存储
    • utf-8: Unicode 实现方式之一，变长编码，用 1$\sim$4 个字节编码一个字符。

      编码方式: 单字节符号第一位设 $0$，$n$ 字节符号 $(n>1)$ 的前 $n$ 位设 $1$，第 $n+1$ 位设 $0$; $10$ 用作非首字节标识符
      • 相比于 utf-16，兼容 ascii，英文字符只需要 1 字节；多数汉字需要 2 个字节，多占用了一个字节。因此对于英文较多的场景比较省空间。

    (2). 表示

    从 python3 开始，str 类型代表 Unicode 字符串，具体的存储方式是根据字符串内容变化的。 

    alphabet = 'abc'
alphabet_eq = u'abc'
    bytes 是 python3 新增的字节串类型，即一个表示字节流的数组。与之相关的方法:
    str.encode(encoding="utf-8", errors="strict") 返回编码后的字节串
    bytes.decode(encoding="utf-8", errors="strict") 返回解码后的字符串
    需要存储文件或网络传输时，需要用到这些方法。 
    # create bytes
s1 = bytes('hello', encoding='ascii')
s2, s3, s4 = b'a', b'\xe4', b'\u54c8'

    (3). 运算符

    a = 'hello world'
# concatenation
print(a + " ", a * 3)

# cut
print(a[:2], a[-1])

    3. 显式强制转换

    通过 python 内建函数实现。

    int(x, base=10), float(x), str(x) 可转换 str 和其他数字类型
    repr(x) 抽象意义上，比 str(x) 更正式，包含更多关于对象的信息
    eval(str) 执行字符串表达式，返回结果
    ord(x), chr(x) 返回字符的 Unicode 值, 返回值的 Unicode 字符
    hex(x), oct(x) 进制转换
