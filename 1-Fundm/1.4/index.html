    4.Dict

    1. 初始化与访问

     字典也是一种可变容器，等同于其他语言中的 map。key 与 key 以及 value 与 value 之间的类型可以不同。

    1
2
3
    dict1 = {'a': 1, 2: 'b'}
print(dict1['a'])
print(dict1[2])
     可以通过访问直接修改 value 或添加 key-value。 
    1
2
    dict1[2] = 3
dict1['ff'] = 'surr'
     如果访问的 key 不存在，会抛出一个 KeyError。因此访问前，可以通过操作符 in 或 get() 方法事先判断。
     在 Python 中，dict 的 key 必须是不可变对象，因此可以是数、字符串、元组，而不能是列表。

    dict 的构造函数：

    • dict()
    • dict(**kwargs)
    • dict(mapping)
    • dict(iterable)

    # empty dict
dict1 = dict()

# init by assignment expression
dict2 = dict(a = 'a', b = '1') # eq: {'a': 'a', 'b': 1}

# init by mapping
dict3 = dict(zip([1, 2, 3], ['a', 'b', 'c']))

# init by iterable object
dict4 = dict([(1, 'a'), [2, 'b']])

    2. Dict 函数

    函数名 作用
    len(seq) 返回键值个数
    list(seq)、tuple(seq) 把键的序列转换为列表或元组
    type(var) 返回输入的变量所属的类型

    3. Dict 方法

    方法名 作用
    get(key, default=None) 返回对应 key 对应的 value，如果不存在返回 default
    has_key(key) 判断是否有 key
    setdefault(key, default=None) 类似于 get()，额外地，如果 key 不存在于字典中，向字典中添加 (key->default)
    pop(key[, default]) 删除字典中 key 对应的 value 并返回被删掉的值。如果 key 不存在且设置了 default，返回 default；否则抛出 KeyError
    keys(),values(),items() 返回可遍历的键、值、键值的列表
    fromkeys(seq[, val]) 创建一个新字典，以 seq 为键值序列，val 为该字典所有值的初始值
    update(dict) 把 dict 中的键值对都更新到字典里
    clear() 清空所有元素
    copy() Shallow Copy
