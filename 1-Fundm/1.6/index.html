    2. 循环结构

    (1). 基本循环结构

    Python 中有 while 和 for-in 两种循环结构。for 循环的结构为: 

    1
2
    for <var> in <iterable>:
    <statement>

    如果需要按照索引遍历，可以使用 range() 函数：

    range(start=0, stop, step=1) 创建一个整数 List，步长为 step，范围为 [start, stop)

    (2). else 语句

    Python3 支持 while-else 和 for-else：

    • while-else: 当且仅当循环因为条件退出，而没有被 break，退出时执行一次 else 块
    • for-else: 当且仅当循环因为遍历结束退出，而没有被 break，退出时执行一次 else 块

    3. 缩进规则

     Python 通过缩进划分作用域。每行代码都依它的缩进格数有一个缩进级别，Python 解释器维护一个缩进栈，在读取 py 文件前，首先把缩进级别 0 压入栈中。然后从头到尾，读取每行代码，当读取到一行有效代码的缩进级别时，与栈顶值比较：

    • 如果相等，什么也不会发生；
    • 如果大于栈顶值，该行代码的缩进级别入栈，同时生成一个缩进标记；
    • 如果小于栈顶值，那么栈中所有比该行代码的缩进级别大的值都会被移除，直到找到一个与自己的级别相等的值，同时生成一个扩展标记。
