    3.List and Tuple

    1. 初始化与访问

     List 是 Python 中最基本的容器。类似于一些其他语言，它可以动态边长，其中可以存储各种对象。

    1
2
    empty_list = []
list1 = [1, 'beginner']
     其索引从 0 开始，可以通过下标直接访问，类似于字符串，也可以通过 : 截取、通过负的索引从后向前访问。

     Tuple 是不可变的 List：一旦创建，其存储的每个元素的值不可改变。 

    1
2
    tuple1 = ('dayup')
tuple2 = (2,)
     当元组中只有一个数字的时候，为避免歧义，应该在第一个元素后加逗号。

    2. List/Tuple 函数

    函数名 作用
    len(seq) 返回元素个数
    max(seq) 返回元素最大值 (元素类型相同)
    min(seq) 返回元素最小值 (元素类型相同)
    list(seq) 转换为 list
    tuple(seq) 转换为 tuple
    type(var) 返回输入的变量所属的类型
    sorted(iter_obj, cmp=None, key=None, reverse=false) 返回一个排序的可迭代对象

    3. List 方法

    方法名 作用
    append(obj) 在列表末尾添加对象
    insert(index, obj) 将对象插入到指定位置，如果index指定的位置未开辟则无效
    pop([index = -1]) 删除列表中元素。当不指明索引时默认为-1
    remove(obj) 移除第一个 obj 的匹配项
    index(obj, [start, [end]]) 返回某个元素在列表中第一次出现的索引，也可以指明查找的范围
    count(obj) 返回某个元素在列表中出现的次数
    reverse() 反转列表
    extend(seq) 在列表末尾追加另一个序列
    sort(cmp=None, key=None, reverse=false) 把列表排序
    Tuple 则只支持其中的两个 Observer: index()、count()。

    4. List 操作符

    操作符 作用
    [1, 2] + [3, 4] 拼接
    ['hello'] * 6 重复拼接
    1 in [1, 2, 3] 存在于
    for x in [1, 2] 迭代
    以上所有运算符也适用于 Tuple。
