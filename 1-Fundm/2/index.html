    2. containers

    1. list and tuple

    当 tuple 初始化只有一个元素时，为避免歧义要加逗号

    (1). function

    函数名 作用
    len(seq), type(var) ...
    max(seq), min(seq) 最大最小值 (元素类型相同)
    list(seq), tuple(seq) 强制转换
    sorted(iterable, *, key=None, reverse=False) 保留原对象，返回一个排好序的可迭代对象 
    students = [('john', 'A', 15), ('jane', 'B', 12), ('dave', 'B', 10)]
sorted(students, key=lambda s: s[2], reverse=True)

    (2). method

    方法名 作用
    append(obj) 追加对象
    insert(index, obj) 插入到指定位置，若 index 指定的位置未开辟则无效
    pop(index=-1) 删除元素
    remove(obj) 移除第一个 obj 的匹配项
    index(obj, [start, [end]]) 返回某元素第一次出现的索引，也可指明查找的范围
    count(obj) 返回某元素出现次数
    reverse() 反转
    extend(seq) 在列表末尾追加另一个序列
    sort(*, key=None, reverse=False) 在原列表上排序

    tuple 则只支持其中的两个 Observer: index()、count()。

    (3). operator

    操作符 作用
    [1, 2] + [3, 4], ['hello'] * 6 拼接
    1 in [1, 2, 3] 存在于
    for x in [1, 2] 迭代

    2. dict

    key 必须 immutable

    (1). ctor

    a = {'one': 1, 'two': 2, 'three': 3}
b = dict(one=1, two=2, three=3)
c = dict([('two', 2), ('one', 1), ('three', 3)])
d = dict(zip(['one', 'two', 'three'], [1, 2, 3]))
e = dict({'three': 3, 'one': 1, 'two': 2})
print(a == b == c == d == e)

    (2). method

    方法名 作用
    get(key, default=None) 返回对应的 value, 不存在则返回 default
    has_key(key) 在 python3 中已被 in 代替
    setdefault(key, default=None) 存在对应的 value 则返回，否则插入 key-default
    pop(key[, default]) 删除 key 对应的 value , 并返回被删掉的值。
    若 key 不存在，返回 default；若 default 也未设置，抛出 KeyError
    keys(), values(), items() 返回可遍历的 键, 值, 键-值 的列表
    fromkeys(iterable[, val]) classmethod.
    以 iterable 中的元素为键创建新字典，值全设为 val (默认 None)
    update(dict) 把 dict 中的 k-v 都更新到字典里，已存在的会被覆盖
    clear() 清空所有元素
    copy() Shallow Copy

    3. set

    创建空集只能通过 set()，{ } 代表空 dict

    (1). ctor

    set([iterable])
    frozenset([iterable])

    (2). method

    方法名 作用 运算符版本
    isdisjoint(S) 判断两集合是否不相交 \
    issubset(other), issuperset(other) 是否为指定集合的子/超集 set <= other
    (< → <= and !=)
    union(*others) 并 set | other | ...
    intersection(*others) 交 set & other & ...
    difference(*others) 差 set - other - ...
    symmetric_difference(other) 对称差 set ^ other
    update(*others) 添加新的元素或集合到此集合中 set |= other | ...
    intersection_update(*others) 交 (在原集合上删除非公共元素) set &= other & ...
    difference_update(*others) 差 (在原集合上移除) set -= other | ...
    symmetric_difference_update(other) 对称差 (原集上操作) set ^= other
    add(e) 添加一个元素 \
    discard(e)) 若存在则删除 \
    remove(e) 删除，不存在则报 KeyError \
    pop() 随机移除一个元素，为空则报 KeyError \

    frozenset 只支持前 6 个方法
